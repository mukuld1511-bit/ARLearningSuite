# ARLearningSuite

## About

A modular, Unity-based Augmented Reality platform designed to transform traditional learning into immersive, interactive experiences. ARLearningSuite is the reusable backbone behind multiple educational AR modules — built and led as part of the **AR/VR Studio Lab at PIET**, with a 6-member development team.

## ✨ Features

- 🧩 Modular, plug-and-play architecture — drop in a new subject module without rebuilding the core
- 🎯 Image-target & plane-detection based AR tracking
- 👆 Tap-to-spawn and pinch-to-zoom interactions
- 💡 Light estimation for realistic rendering
- 📱 Built and optimized for Android tablets (e.g., Samsung Galaxy Tab S6 Lite)

## 📚 Modules Built on This Suite

- Human Skeleton
- Human Brain
- Wild Animals
- Taj Mahal
- Dinosaurs
